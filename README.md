# Admin Dashboard (MERN Stack)

A full-stack admin dashboard built using **Node.js, Express, MongoDB, and React** with JWT authentication and role-based access control.

## Features
- JWT-based Authentication
- Role-based Authorization (Admin/User)
- Product Management (CRUD)
- Order Management
- Sales Report using MongoDB Aggregation
- Protected Routes (Frontend)
- Axios Interceptors for Token Handling

## Tech Stack
**Backend**
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT

**Frontend**
- React.js
- React Router
- Axios
- Context API

## API Modules
- Auth
- Users
- Products
- Orders
- Reports (Aggregation)
